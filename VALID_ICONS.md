## How to Check if Icon Exists

1. Go to: https://lucide.dev/icons
2. Search for the icon name
3. Use exact name from the website

## Common Replacements

If an icon doesn't exist, use these alternatives:

- `IdCard` → `CreditCard` or `Badge`
- `UserPen` → `Edit` or `UserCog`
- `Profile` → `User` or `UserCircle`
- `Document` → `FileText` or `File`
- `Folder` → `FolderOpen` or `Folder`
